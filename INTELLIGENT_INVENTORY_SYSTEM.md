# Intelligent Inventory Decision System

> A rule-based AI inventory intelligence platform for SMEs — built on FastAPI, React, and PostgreSQL. Delivers real-time stock insights using deterministic SQL algorithms with zero dependency on paid AI or ML services.

## Features

| Feature | Description |
|:--------|:------------|
| Stockout Risk Scoring | Bounded 0–100 score per product using deterministic SQL formulas |
| Demand Trend Classification | Rising, Stable, or Declining based on 30-day rolling comparison |
| IntelliReorder™ | Automated reorder quantity suggestions triggered at reorder level |
| Inventory Turnover | Fast, Moderate, or Slow classification per product |
| Anomaly Detection | Flags demand spikes and drops exceeding 50% deviation |
| Days-Until-Stockout | Concrete countdown per product based on current sales velocity |
